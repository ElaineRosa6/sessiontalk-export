# 使用说明

## 快速开始

### 1. 安装插件

#### Chrome 插件方式

1. 构建项目：`npm install && npm run build`
2. 打开 Chrome，访问 `chrome://extensions/`
3. 启用「开发者模式」（右上角开关）
4. 点击「加载已解压的扩展程序」
5. 选择项目的 `dist` 目录

#### 油猴脚本方式

1. 安装 Tampermonkey 浏览器扩展
2. 打开 Tampermonkey 管理面板
3. 点击「添加新脚本」
4. 粘贴 `userscript/zhipu-exporter.user.js` 内容
5. 保存并启用

### 2. 使用插件

1. 打开智谱清言网页 https://chatglm.cn
2. 登录你的账号
3. 点击浏览器右上角的插件图标

### 3. 导出对话

#### 单条导出

1. 在对话列表中找到目标对话
2. 勾选该对话
3. 选择导出格式
4. 点击「导出选中对话」

#### 批量导出

1. 使用「全选」或手动勾选多个对话
2. 选择导出格式
3. 选择是否合并为一个文件
4. 点击「导出选中对话」

## 导出格式对比

### Markdown (.md)

```markdown
# 对话标题

> 导出时间: 2024-01-15 10:30:00

---

### 👤 用户

用户消息内容...

---

### 🤖 助手

AI 回复内容...

---
```

**优点**：
- 保留代码块格式
- 支持大多数笔记软件
- 易于阅读和编辑

**适用场景**：Obsidian、Notion、Typora、博客

### 纯文本

```
对话标题: 对话标题
导出时间: 2024-01-15 10:30:00

[用户]:
用户消息内容...

------------------------------

[助手]:
AI 回复内容...

------------------------------
```

**优点**：
- 无格式干扰
- 文件体积小

**适用场景**：简单存档、文本处理

### JSON (.json)

```json
{
  "id": "conv_xxx",
  "title": "对话标题",
  "timestamp": 1234567890,
  "messages": [
    {
      "role": "user",
      "content": "用户消息内容",
      "timestamp": 1234567890
    },
    {
      "role": "assistant",
      "content": "AI 回复内容",
      "timestamp": 1234567890
    }
  ]
}
```

**优点**：
- 结构化数据
- 便于程序处理

**适用场景**：数据分析、二次开发

### HTML (.html)

离线可读的网页文件，保留基本样式。

**优点**：
- 可在浏览器直接查看
- 保留视觉样式

**适用场景**：网页查看、分享

## 设置选项

点击插件弹窗右上角的 ⚙ 图标进入设置页面。

### 可配置项

| 选项 | 说明 | 默认值 |
|------|------|--------|
| 默认导出格式 | 快速导出使用的格式 | Markdown |
| 包含时间戳 | 消息是否包含时间信息 | 是 |
| 包含元数据 | 对话 ID、模型信息等 | 是 |
| 文件命名模板 | 导出文件命名规则 | `{title}_{date}` |

## 常见问题

### Q: 对话列表显示为空？

可能原因：
1. 页面未完全加载
2. DOM 选择器需要更新

解决方案：
1. 刷新智谱清言页面
2. 参考 `docs/API-ANALYSIS.md` 更新选择器

### Q: 导出内容不完整？

可能原因：
1. 对话包含特殊内容（图片、文件）
2. 需要进入对话详情页获取完整内容

解决方案：
1. 先进入对话详情页再导出
2. 等待完整内容加载

### Q: 导出文件名乱码？

检查对话标题是否包含特殊字符，插件会自动替换为下划线。
